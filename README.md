# About the project

Django + Nuxt E-Commerce demo app

This project is broken into a Django/Rest-Framework rest api and Nuxt frontend.

## Features

* [x] User management
* [x] Integrated payment system
* [x] Order history
* [x] Dynamic filters
* [x] Responsive Design

## Technologies
* Django
* Nuxt.js
